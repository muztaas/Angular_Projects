This is a Weather Forecast application which fetches the weather data from open-meteo which takes the latitude and the longitude to provide the temperature, humidity, and weather conditions according to the query string provided.

This application also has a page for which consumes an API Geoapify which provides the longitude and latitude of a city.

Both the APIs are used to get the weather forecast of a city by using the first API to get the latitude and longitude of a user entered city and to get the weather forecast of that latitude and longitude by consuming the second API.
